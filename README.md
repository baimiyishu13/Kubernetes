K8s 为我们带来了什么？

1. 拜托繁琐的工作：一键式创建一整套环境；
2. 降低故障频率：传统架构、很多基础问题如基础环境、依赖版本等问题，处理告警的繁琐工作；K8s使用容器部署；
3. 简化服务配置：简单的额皮质实现非常复杂的功能；
4. 更加强大的服务管理：自动扩缩容、更灵活的服务调度等；

> K8s的学习始终是一个难题，许多人只知其表不知其里
